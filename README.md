## ✨ Features

### 👨‍💼 Employee & Department Management
- Add, update, and delete employees
- Manage departments efficiently

### 📊 Dashboard Analytics
- Displays total number of employees
- Displays total number of departments
- Provides a quick overview of system data

### 📋 Task Management System
- Managers can assign tasks to employees
- Set deadlines for each task
- Track task status:
  - Pending
  - In Progress
  - Completed

### 📥 Excel Report Export
- Download employee data in Excel format
- Includes:
  - Employee Name
  - Department Name
  - Assigned Task
  - Deadline
  - Task Status

### 🔗 REST API Integration
- Backend APIs built using Spring Boot
- Seamless integration with React frontend
